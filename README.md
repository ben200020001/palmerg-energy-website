# Palmerg Energy — Website

React (Vite) frontend with an Express API (`server/index.js`) for contact email, AI-assisted search, and production static hosting from `dist/`.

## Quick start

```bash
npm install
npm run dev
```

- **App:** http://localhost:5173  
- **API:** http://localhost:5174 — proxied to `/api/*` from Vite in dev  

See **README-RUN.md** for environment variables and production (`NODE_ENV=production`).

## Repo

Push changes to your Git remote; deploy (e.g. Render, Fly.io) from this repository using your platform’s build/start commands.
